# Smart Bakery System (Java)

## 📌 Description
This is a simple Smart Bakery System developed in Java as a Programming Fundamentals project. It manages basic bakery operations like orders and items.

## 🚀 Features
- Add bakery items
- Manage orders
- Display items

## 🛠 Technology
- Java

## ▶️ How to Run
1. Compile:
   javac src/BakerySystem.java
2. Run:
   java src.BakerySystem

Project report is available in the `docs` folder.
